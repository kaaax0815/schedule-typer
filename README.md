```json
{
  "bushaltestelle": "TEst123",
  "zeiten": ["08:15", "|", "NULL", "08:55"]

```

Type Bus Schedules easier

Options:
  -V, --version  output the version number
  -A, --about    About
  -S, --start    Start the Input
  -h, --help     display help for command
